Davidup — ball-showcase (every composability primitive · one ball.png)

Same composition as the node MP4 demo at examples/ball-showcase-60s/: a root file that $refs five scene definitions and a crossfade snippet, defines a user template labeledBall, uses every built-in template (titleCard · lowerThird · bulletList · kenburnsImage · captionBurst) and every built-in behaviour (fadeIn / fadeOut / popIn / popOut / slideIn / slideOut / rotateSpin / kenburns / shake / colorCycle / pulse). The browser driver runs precompile() first, then attaches a RAF loop — exactly the same canonical JSON the MP4 path consumes.
