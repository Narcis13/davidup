# davidup project

A new davidup project. Open it in the editor with:

```bash
davidup edit .
```

The `composition.json` file at the root is the canonical source of truth. The
editor reads and writes this file directly — keep it under version control.

## Layout

- `composition.json` — the canonical composition document.
- `library/index.json` — local library catalog (templates, behaviors, scenes,
  assets, fonts).
- `assets/` — project assets (images, audio, fonts).
- `renders/` — output files (gitignored by default).
