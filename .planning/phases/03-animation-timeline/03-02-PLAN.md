<task type="auto">
  <name>Task 1: Create Animation Schemas</name>
  <files>src/schemas/animation.ts, tests/schemas/animation.test.ts</files>
  <action>
Create src/schemas/animation.ts with these Zod schemas:

1. EasingNameSchema - z.enum with all 12 easing names:
   'linear', 'easeIn', 'easeOut', 'easeInOut',
   'easeInCubic', 'easeOutCubic', 'easeInOutCubic',
   'easeInBounce', 'easeOutBounce', 'easeInOutBounce',
   'easeInElastic', 'easeOutElastic', 'easeInOutElastic'

2. KeyframeSchema - z.object:
   - time: z.number().min(0) - time in seconds from element start
   - value: z.number() - the value at this keyframe
   - easing: EasingNameSchema.optional() - easing to use when interpolating TO this keyframe

3. PropertyAnimationSchema - z.object:
   - property: z.enum(['x', 'y', 'rotation', 'scaleX', 'scaleY', 'opacity'])
   - keyframes: z.array(KeyframeSchema).min(1)

4. AnimationPresetSchema - z.object:
   - type: z.enum(['fade', 'slide', 'scale', 'bounce'])
   - duration: z.number().positive().default(0.5)
   - direction: z.enum(['left', 'right', 'top', 'bottom']).optional()
   - distance: z.number().positive().optional()
   - easing: EasingNameSchema.optional()

5. TransitionSchema - z.object:
   - type: z.enum(['fade', 'slide', 'zoom'])
   - duration: z.number().positive().default(0.5)
   - direction: z.enum(['left', 'right', 'up', 'down']).optional()
   - easing: EasingNameSchema.optional()

Export inferred types: Keyframe, PropertyAnimation, AnimationPreset, Transition, EasingName

Create tests/schemas/animation.test.ts with:
- Valid keyframe parsing
- Keyframe with negative time rejected
- Property animation with empty keyframes rejected
- Valid preset parsing with defaults
- Transition schema validation
  </action>
  <verify>npm test -- tests/schemas/animation.test.ts</verify>
  <done>All animation schemas export and validate correctly</done>
</task>
