<feature>
  <name>Easing Functions</name>
  <files>src/animation/easing.ts, src/animation/types.ts, tests/animation/easing.test.ts</files>
  <behavior>
    All easing functions follow signature: (t: number) => number
    Where t is normalized time 0-1, returns eased progress.

    Expected values at key points:

    linear:
      t=0 -> 0, t=0.5 -> 0.5, t=1 -> 1

    easeIn (quadratic):
      t=0 -> 0, t=0.5 -> 0.25, t=1 -> 1

    easeOut (quadratic):
      t=0 -> 0, t=0.5 -> 0.75, t=1 -> 1

    easeInOut (quadratic):
      t=0 -> 0, t=0.5 -> 0.5, t=1 -> 1 (but curve is S-shaped)

    easeInCubic:
      t=0 -> 0, t=0.5 -> 0.125, t=1 -> 1

    easeOutCubic:
      t=0 -> 0, t=0.5 -> 0.875, t=1 -> 1

    easeInOutCubic:
      t=0 -> 0, t=0.5 -> 0.5, t=1 -> 1

    easeOutBounce:
      t=0 -> 0, t=1 -> 1 (bounces at end)

    easeInBounce:
      t=0 -> 0, t=1 -> 1 (bounces at start)

    easeInOutBounce:
      t=0 -> 0, t=0.5 -> 0.5, t=1 -> 1 (bounces both ends)

    easeOutElastic:
      t=0 -> 0, t=1 -> 1 (may overshoot past 1)

    easeInElastic:
      t=0 -> 0, t=1 -> 1 (may undershoot below 0)

    easeInOutElastic:
      t=0 -> 0, t=1 -> 1 (oscillates)

    getEasingFunction('easeIn') -> returns easeIn function
    getEasingFunction('invalid') -> returns linear (fallback)
  </behavior>
