2. Create `studio/src/stores/chatStore.ts` with Zustand store:

Interface for Message:
```typescript
interface Message {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  template?: object;  // Extracted JSON template if present
  createdAt: string;
}
```

Store state:
- `conversationId: string | null` - Current conversation ID
- `messages: Message[]` - Messages in current conversation
- `currentStreamingMessage: string` - Content being streamed (accumulates chunks)
- `isGenerating: boolean` - True while AI is generating
- `error: string | null` - Error message if request failed

Store actions:
- `setConversationId(id: string | null)` - Set current conversation
- `addUserMessage(content: string)` - Add user message to array
- `appendToCurrentMessage(chunk: string)` - Append chunk to streaming message
- `finalizeAssistantMessage(template?: object)` - Move streaming content to messages array
- `setIsGenerating(value: boolean)` - Set generating state
- `setError(error: string | null)` - Set error state
- `clearConversation()` - Reset all state for new conversation
- `loadConversation(id: string, messages: Message[])` - Load existing conversation

Use `crypto.randomUUID()` for message IDs and `new Date().toISOString()` for createdAt.

1. Create `studio/src/hooks/useChatStream.ts`:
   - Uses fetch + ReadableStream to consume SSE (not EventSource - can't POST)
   - Accepts `{ conversationId, message, history }` options
   - Uses AbortController for cleanup (stores in ref)
   - Parses SSE format: `data: {...}\n\n`
   - Calls store actions: `setIsGenerating`, `appendToCurrentMessage`, `finalizeAssistantMessage`, `setError`
   - Returns `{ sendMessage, abort }` functions
   - Handle 'chunk', 'done', and 'error' event types

2. Create `studio/src/hooks/useCopyToClipboard.ts`:
   - Uses `navigator.clipboard.writeText()`
   - Returns `{ copy: (text: string) => Promise<boolean>, isCopied: boolean }`
   - Auto-resets `isCopied` to false after 2 seconds

3. Create `studio/src/hooks/useAutoScroll.ts`:
   - Returns `{ containerRef, isAtBottom, scrollToBottom }`
   - Tracks scroll position with scroll event listener
   - Consider "at bottom" if within 100px of bottom
   - `scrollToBottom()` uses `scrollTo({ top: scrollHeight, behavior: 'smooth' })`

4. Create `studio/src/api/chat.ts` with TanStack Query hooks:
   - `useConversations()` - Fetches GET /studio/conversations
   - `useConversationMessages(conversationId)` - Fetches GET /studio/conversations/:id/messages
   - `useDeleteConversation()` - DELETE mutation with query invalidation

5. Update `studio/src/main.tsx`:
   - Import `QueryClient`, `QueryClientProvider` from @tanstack/react-query
   - Create QueryClient instance
   - Wrap App in QueryClientProvider
