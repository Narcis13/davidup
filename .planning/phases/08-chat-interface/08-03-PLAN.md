   - Get state from useChatStore: messages, currentStreamingMessage, isGenerating, error, conversationId
   - Get actions from useChatStore: addUserMessage, clearConversation, setConversationId
   - Get sendMessage from useChatStream hook

   Handle message sending:
   ```typescript
   const handleSend = async (message: string) => {
     // Add user message to store immediately (optimistic)
     addUserMessage(message);

     // Build history from current messages
     const history = messages.map(m => ({ role: m.role, content: m.content }));

     // Send to backend
     await sendMessage({
       conversationId: conversationId ?? undefined,
       message,
       history,
     });
   };
   ```

   Listen for conversation ID from stream (via store update in useChatStream).
