<feature>
  <name>Variable Substitution</name>
  <files>src/api/services/variable-substitution.ts, tests/api/services/variable-substitution.test.ts</files>
  <behavior>
**extractVariables(spec: VideoSpec): string[]**
- Finds all {{variableName}} patterns in the entire spec
- Returns unique variable names (no duplicates)
- Returns empty array if no variables found
- Handles nested objects and arrays

Cases:
- { text: "Hello {{name}}" } -> ["name"]
- { text: "{{greeting}}, {{name}}!" } -> ["greeting", "name"]
- { scenes: [{ text: "{{title}}" }, { text: "{{title}}" }] } -> ["title"] (deduplicated)
- { text: "No variables" } -> []
- { nested: { deep: { text: "{{value}}" } } } -> ["value"]

**substituteVariables(spec: VideoSpec, values: Record<string, string>): VideoSpec**
- Replaces all {{variableName}} with corresponding values
- Escapes special JSON characters in values (quotes, backslashes, newlines)
- Leaves unmatched {{variables}} as-is (partial substitution)
- Returns valid VideoSpec (JSON round-trip safe)

Cases:
- substituteVariables({ text: "{{name}}" }, { name: "World" }) -> { text: "World" }
- substituteVariables({ text: "{{name}}" }, {}) -> { text: "{{name}}" } (unchanged)
- substituteVariables({ text: "{{name}}" }, { name: 'Say "Hello"' }) -> { text: 'Say "Hello"' } (escaped quotes)
- substituteVariables({ text: "{{a}} and {{b}}" }, { a: "X" }) -> { text: "X and {{b}}" } (partial)
  </behavior>
  <implementation>
1. extractVariables:
   - Serialize spec to JSON string
   - Use regex /\{\{(\w+)\}\}/g to find all matches
   - Collect unique variable names in Set
   - Return sorted array

2. substituteVariables:
   - Serialize spec to JSON string
   - For each key in values: escape JSON special chars, then replace
   - Parse back to object
   - Return typed result

Edge cases to handle:
- Empty spec
- Values containing quotes, backslashes, newlines, unicode
- Variables that don't match (leave as-is)
- Duplicate variables get same value
  </implementation>
</feature>
